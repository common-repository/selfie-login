Log in to Wordpress profile with your selfie!

== Description ==

SmilePass Selfie Login is a plugin allowing you to log in to your Wordpress profile just by capturing your selfie. It's very fast and convinient face authentication solution provided by SmilePass Ltd. 

**Features:**

* Fast log in to Wordpress profile using selfie
* Both HTML5 & Flash are supported
* Facial recognition performed over secure connection provided by SmilePass Ltd
* Configurable security levels (in the admin panel)
* Easy setup process

Please note:

1. Selfie Login works on iOS devides using Puffin browser only.
1. For Android phones, your website should support https secure connection.
**Those issues will be fixed in the upcoming releases.**

In order to use Selfie Login, you should have an account in SmilePass system. You can create it by registering inside the plugin setting in the Wordpress Admin panel. After the trial period (5000 API calls), you will manage your SmilePass profile at https://subscriber.smile-pass.com/login.

== Installation ==

**Via your Wordpress Dashboard:**

1. Log in to your Wordpress dashboard, go to Plugins >> Add New, search for Selfie Login, and click to install.
1. Activate the plugin through the 'Plugins' screen in WordPress.
1. Use the Settings->"Smilepass plugin global settings" screen to configure the plugin.
1. Fill in the registration form and after fill in the authorization form with your credentials. 
1. Ensure Served forms->Login form is checked.

**Via FTP:**

1. Upload the plugin files to the `/wp-content/plugins/smilepass` directory
1. Activate the plugin through the 'Plugins' screen in WordPress.
1. Use the Settings->"Smilepass plugin global settings" screen to configure the plugin.
1. Fill in the registration form and after fill in the authorization form with your credentials. 
1. Ensure Served forms->Login form is checked.

= How does the Selfie Login work? =
The plugin uses SmilePass Biometric Engine, that uses complex matching algorithms by measuring facial features, and aspects that are only identifiable, on a live human face.
